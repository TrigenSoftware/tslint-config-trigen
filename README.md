Trigen's TSLint config.

## Install

```bash
npm i -D tslint-config-trigen
# or
yarn add -D tslint-config-trigen
```

## Configure

Add `'extends': 'trigen'` (to lint code with ReactJS) or `'extends': 'trigen/base'` to your `tslint.json`.
